## âš™ï¸ Installation

### Prerequisites
- **Node.js** and **npm** installed on your machine.

### Setup
1. Clone the repository:
    ```bash
    git clone https://github.com/makhalifa/multiplayer-pong.git
    cd multiplayer-pong
    ```
2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the server:
    ```bash
    npm start
    ```

4. Open your browser and navigate to:
    ```
    http://localhost:4000
    ```
